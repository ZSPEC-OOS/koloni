Tool settings
Timeout 5 sec
How long a tool can run before it's automatically stopped. Shorter = faster failures, longer = more patience.
Max Retries 3×
If a tool fails, how many times should the system try again before giving up?
Token Budget 5,000
The swarm-wide limit on how many tokens can be used. Shared across all agents.
Safety rules (always enforced)
🚫 Network + Code Execution
A tool can't access the internet AND run code at the same time — prevents data from being secretly sent out.
🚫 File Write + Spawn Sub-Colony
Can't write files and spawn new agents together — prevents one agent from poisoning others.
🚫 Research + Email Send
Can't look things up and send email in one step — prevents automated phishing.
