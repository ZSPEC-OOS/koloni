ContextManager
Shared state with scoping & inheritance

Scope Descriptions
GLOBAL: Swarm-wide (security policies, token budgets)
COLONY: Task-specific (intermediate results)
AGENT: Agent workspace (capabilities, claims)
PRIVATE: Tool-only (auto-cleanup)
Context Operations
get(key, scope?): Read with inheritance → {value, scope}
getValue(key): Read raw value (convenience)
set(key, value, scope): Write to specific scope
fork(): Immutable snapshot
merge(snapshot): Restore from snapshot
