ContextManager
Shared state with scoping & inheritance
🌍 global.memory shared
🏛️ colony.config inherited
🤖 agent.state isolated
🔒 tool.scratchpad private
Context Scopes
GLOBAL: Swarm-wide shared memory (token budgets, security policies)
COLONY: Task-specific shared state (intermediate results)
AGENT: Agent-private workspace (sub-colony isolation)
PRIVATE: Tool-only temporary storage (auto-cleanup)
Context Operations
get(key, scope?): Read with inheritance
set(key, value, scope): Write to specific scope
fork(): Immutable snapshot for rollback
merge(snapshot): Restore from snapshot
